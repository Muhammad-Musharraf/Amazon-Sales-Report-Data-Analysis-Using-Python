# 📊 Amazon Sales Report Data Analysis Using Python

## 📖 Project Overview

This project focuses on analyzing Amazon sales data using Python to uncover valuable business insights and sales trends. The analysis involves data cleaning, preprocessing, exploratory data analysis (EDA), and data visualization to understand customer behavior, product demand, fulfillment methods, and regional sales performance. The project utilizes Python's powerful data analysis libraries to transform raw sales data into meaningful business intelligence.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Analyze Amazon sales performance.
* Identify top-selling product categories.
* Examine customer purchasing patterns.
* Analyze fulfillment methods and shipment data.
* Discover regional sales trends.
* Create informative visualizations for business insights.

---

## 📊 Dataset Description

The dataset contains Amazon sales transaction records with information such as:

* Order Date
* Product Category
* Size
* Quantity
* Fulfillment Method
* Shipment Status
* State and City Information
* B2B Orders
* Courier Status
* Sales Channel

The dataset enables detailed analysis of customer behavior and sales performance across different dimensions.

## 📈 Key Insights

### 👕 Product Category Analysis

* T-Shirts emerged as one of the most popular product categories among customers.

### 📏 Size Preference Analysis

* Medium (M) size products showed the highest demand among buyers.

### 🏢 Customer Type Analysis

* The majority of purchases were made by retail customers rather than B2B buyers.

### 🚚 Fulfillment Analysis

* Most orders were fulfilled through Amazon's fulfillment services.

### 🌍 Geographic Analysis

* Maharashtra recorded the highest customer concentration among all states analyzed.
